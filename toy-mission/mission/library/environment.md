# Environment Facts — wordstats toy mission

- **bash** is available (macOS default bash 3.2 — avoid `mapfile`, associative arrays, `${var,,}`).
- **awk** is available (BSD awk on macOS) — the counting logic uses it.
- **git** is available; the scratch repo is git-initialized with an initial scaffold commit.
- No other tools may be assumed and none may be installed. No network access is needed.
- The repo root is the directory containing `mission/`; run all commands from there.
